# Smart-Groceries
Een slimme boodschappen lijst

Smart groceries heeft als doel om de beste prijzen van verschillende
winkels te vergelijken en jou geld te besparen.


Hierna volgen een paar simpele stappen om het project te installeren:

1. Download en installeer JDK 21
    https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html

2. Ga naar de git repository folder

3. Open een command line window en run de volgende command om 
    ```bash id="x8k2mq"
    .\mvnw.cmd package
    ```
4. Build de docker container
    ```bash id="x8k6mq"
    docker compose up --build
    ```
